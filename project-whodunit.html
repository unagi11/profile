실감형 1인칭 추리 게임 Co-op 실감형 게임 콘텐츠 개발트랙 (2021-1)
Unity C# Shader Graph

Overview

논리적 추론과 단서만으로 살인 사건을 추리하는 1인칭 추리 게임. 공간 내 오브젝트와 상호작용하며 단서를 수집하고, 본인이 기획한 트릭을 플레이어가 역으로 풀어내는 구조로 설계했습니다.

숭실대학교 2021-1학기 ‘Co-op 실감형 게임 콘텐츠 개발트랙’ 기말과제로, 메인 기획과 주요 프로그래밍을 담당했습니다.
